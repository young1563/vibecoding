# Zootopia 2: Mahjong Master

주토피아 2의 세계관을 배경으로 한 **솔리테어 마작 (Mahjong Solitaire) 퍼즐 게임**입니다.
Vita Mahjong의 중독성 있는 게임플레이를 현대적이고 세련된 주토피아 스타일로 재해석했습니다.

## 📁 프로젝트 구조
- `index.html`: 게임 메인 구조 (스테이지형 마작 레이아웃)
- `style.css`: 3D 입체 타일 디자인 및 프리미엄 글래스모피즘 UI
- `mahjong-game.js`: 마작 엔진 (레이어 생성, 블록 판정, 매칭 로직)
- `data/`: 주토피아 시나리오 데이터 보관

## 🧩 주요 기능
1. **무한 스테이지**: 스테이지가 올라갈수록 타일 배치가 복잡해지고 레이어가 쌓입니다.
2. **3D 타일 시스템**: 타일의 겹침과 좌우 막힘을 계산하여 실제 마작 솔리테어의 룰을 완벽히 구현.
3. **주토피아 캐릭터 인터랙션**: 🐰 주디와 🦊 닉이 게임 상황(매칭, 힌트 없음, 스테이지 클리어)에 맞춰 실시간으로 대화합니다.
4. **특수 스킬**: 셔플(Shuffle), 힌트(Hint) 기능을 통해 막히는 구간을 돌파할 수 있습니다.

## 🦊 게임 방법
1. **가장자리**에 있거나 **위에 가로막는 타일이 없는** 똑같은 모양의 타일 두 개를 선택하세요.
2. 짝을 맞춰 모든 타일을 제거하면 다음 스테이지로 넘어갑니다.
3. 닉과 주디의 조언을 들으며 최고 점수에 도전하세요!
