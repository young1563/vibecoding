# [수사 보고서] 주토피아 수사국(ZIB) 프로젝트 개발 가이드 및 진행 기록

## 1. 프로젝트 비전
**ZIB (Zootopia Investigation Bureau)**: 주토피아의 명수사관 주디와 닉이 되어 사건을 해결하는 하이테크 수사 시물레이션 플랫폼. 
- **컨셉**: 다크 글래스모피즘 기반의 프리미엄 UI, 수사관 전용 장비 활용.
- **주요 게임**: 마작(단서 수집), 노노그램(회로 복구).

---

## 2. 주요 개발 연혁 (Conversation & Prompt History)

### Phase 1: 아키텍처 개편 (SPA -> MPA)
- **사용자 요청**: "페이지 이동으로 구조 변경해줘"
- **조치 사항**:
    - `index.html`, `mahjong.html`, `nonogram.html`로 파일을 분리하여 독립적인 페이지 이동 구조 구축.
    - `PlatformManager`를 개편하여 페이지 간 로컬 스토리지 데이터(별, 점수, 인벤토리) 동기화 구현.

### Phase 2: 마작 수사 모듈 고도화
- **사용자 요청**: "마작 겹쳐지는 것부터 구현 다시 해봐. 버튼 색상 배경이랑 어울리게. 보드판 키워줘."
- **조치 사항**:
    - **입체적 레이어 판정**: 상위 레이어에 패가 있으면 클릭되지 않는 3D 스태킹 로직 구현.
    - **가시성 개선**: 패(Tile) 크기 확대 및 폰트 굵기 강화. 하단 바(Collector) 확장.
    - **매칭 로직 최적화**: 순서에 상관없이 수집 바에 2개만 모이면 즉시 매칭되는 스마트 로직 적용.

### Phase 3: UI/UX 감성 및 가시성 고도화
- **사용자 요청**: "각 버튼 동작 잘되도록. 버튼 설명도 세련되게 넣어줘. 가시성 좋게."
- **조치 사항**:
    - **수사관 도구 설명**: '증거 재배치', '추리 힌트' 등 세련된 한글 라벨 및 데이터 기반 툴팁 적용.
    - **네온 효과**: 버튼 호버 시 주디 블루(`--primary-judy`)와 네온 후광 효과 적용.
    - **툴팁 보정**: 화면 우측 끝에서 문구가 잘리는 버그를 위해 자동 줄바꿈 및 우측 정렬 옵션 추가.
    - **시각적 피드백**: 매칭 성공 시 `match-flash` 애니메이션, 힌트 사용 시 네온 글로우 효과 추가.

---

## 3. 핵심 디자인 가이드 (CSS Tokens)
- **Primary Color**: `#FF8A00` (Nick), `#00B8D4` (Judy)
- **Background**: `#05070A` (Deep Dark)
- **Glow**: `--neon-gold`, `--primary-judy`를 활용한 강조 효과.
- **Component**: `.pill` 클래스를 활용한 고급스러운 버튼 스타일링.

## 4. 향후 확장 추천 사항
- 실시간 수사 랭킹 시스템 (Local Ranking).

- 모바일 전용 수사관 인터페이스(HUD) 최적화.
